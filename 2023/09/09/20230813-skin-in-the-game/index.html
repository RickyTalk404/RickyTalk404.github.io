风险、对称性、Skin in the game

Posted on 2023-09-09 In 杂文
塔勒布关于对称性的认识

纳西姆·塔勒布在《反脆弱》里提到了一个概念叫做「非对称性」，相对的是「对称性」。塔勒布在他的另一本书 Skin in the Game 对于对称性这个概念做了许多诠释，这是个有趣的概念，理解对称性对于理解这个世界有相当大的帮助。

Skin in the game 不太好翻译成中文，直接翻译就是在游戏里有一点皮肤，意思就是和你利益相关。换言之，skin in the game 意味着如果你真的想加入游戏，你需要成为参与者而非空谈家，并为自己行动负责，你是一个 player。

Skin in the game 体现了对称性，你加入游戏，赢了将获利，输了将付出代价。为了享受权利，你要承担义务；为了获得技能，你需要付出艰苦的学习，正如皮肤直接触碰粗糙地面感受疼痛一样, 这种付出是真实世界生存所需。Skin in the game, 你的行为才值得被认真对待。

对称性是自然界广泛存在的原则，人类发展到今天，其实一直都在试图规范风险方面的对称性，或者说纠正非对称性，来维护社会的稳定。如果没有对称性法则, 就会出现把风险转移到无辜者身上的事情，这样做的后果就是增加系统的脆弱性甚至导致系统崩溃。

这里所说的「风险」往往是指「尾部风险」，也就是在概率曲线尾部的风险，以正态分布的钟形曲线为例，尾部就是远离均值的两端，尾部的风险发生的概率很低，但并不为零。如果这个风险发生后将会造成巨大的灾难，那这个风险对应的事件就被塔勒布称作「黑天鹅」。这个风险不容易被看见，往往是因为有人有意无意地增加了系统的脆弱性，当系统发生黑天鹅事件的时候，隐藏的风险所带来的巨大伤害才体现出来，且危及系统内的无辜者，而当初为系统增加脆弱性的人反而可能并不会受到什么损害。

人类的社会从古至今其实一直都在维护对称性。古代王公贵族具有较高的社会地位，往往不是因为他们具有更高的暴力手段，而是因为他们承担了一定的风险和责任。王室为国家承担风险，贵族保护属地内的平民，换来平民为自己纳税，并且享受较高的社会地位，这是一种具有对称性的古老社会契约。

古代历史上，绝大多数的军阀和战争狂人都是亲自上战场的战士，无论是拿破仑、凯撒，还是其他大多数的罗马皇帝，他们都会亲自上战场。古罗马皇帝只有三分之一死在床上，剩下的或死于政变，或战死疆场。这么看来，古代国王也是一个高危职业，欲戴王冠，必承其重，承担巨大的风险，才能当国王。贵族也是如此，战争必须冲锋在前，他们的战死率远高于平民。古代几乎所有的战争都是贵族战争，我认为最后一场贵族战争是一战，作为贵族子弟学校的牛津和剑桥两所大学，参战的学生死亡率分别是 19% 和 18%，远高于平民的死亡率。

当人们以为推翻了王室和贵族，社会就会进步，但其实并不然，社会的进步在于在于保障人类的基本尊严，在于对风险共担的维护，归根结底，进步与否还是在于人们是否能够通过努力纠正非对称性，防止有人把风险转移给无辜的人。从这个角度讲，我们今天的现代社会在一些方面并没有比贵族主导的社会好多少。

在现代国家由官僚制主导的政府中，一个人的行为和他造成的后果被轻易地分开了。这是一种非对称性，如果官僚并不为错误的政策和决策承担相应的责任，发动战争的决策者并不为战争的后果负责，结果就是有人将本来应该承担的风险转移给了无辜的大多数，为整个系统增加了脆弱性，甚至会导致系统崩溃。

最有名的例子应该是2008年的金融危机，银行家和金融系统内的大型参与者，用脱离现实的风险模型，把一系列具有爆炸性风险的资产包装起来从中获利，为系统注入脆弱性。罗伯特·鲁宾是当时的财政部长，任职期间他从花旗银行收获了巨额的回报，而危机发生后政府用纳税人的钱救市，罗伯特·鲁宾从头到尾没有掏过一分钱，而那些银行家们也并没有从自己的转移风险的勾当中受到什么惩罚。这就是最典型的决策者不用承担后果的例子，缺乏相应的对称性风险匹配，既得利益者和决策者就可以不承担风险的情况下获得巨大的利益，而风险会落到无数无辜者的头上。

社会发展出来许多制度来抵消官僚制带来的风，分权制就是其中之一。分权制使决策者的声音只能在有限范围内传播，所造成的影响也仅局限于系统中的一小部分。这样便降低了由风险不匹配导致的非对称性，并进而减少了系统风险。仅通过朴素的对称性原则，我们就能明白分权体制以及对问责机制在控制系统风险方面有着重要作用。而且，对于那些在崩溃过后依然存在的系统，他们都不约而同地走向了分权制从而达到了更稳定的平衡，无论是国家的政治系统，金融系统还是自然界的生物系统。不对称风险积累而造成的崩溃，也可以看作是一种再平衡的机制。

4000多年前的《汉莫拉比法典》有一条著名的法令：如果建筑师造的房子倒了并且导致屋主人死亡那么建筑师将会被处死。古代的许多立法都有着对称性的原则即「以眼还眼，以牙还牙」的同态复仇。当然，当你伤害到别人的时候，过错方未必真的需要付出同样的代价，但一定会有所惩罚。这是维护社会稳定的基本原则。

就对称性原则，塔勒布提到了两条规则，第一条是「金律」，用《圣经》的说法就是「以你想被对待的方式去对待他人」，但这种方式并没有排除这样一种逻辑可能性：你单方面把自己认为对别人有利的事情强加给了别人。所以塔勒布认为银律更加牢固，即「如果你不想要什么，也不要把这个施加给别人」，用中国的一句古话就是「己所不欲，勿施于人」。

银律在许多方面有不同的说法，例如国家对国家之间，如果你认为比你强大的国家应该如何跟你打交道，你就应该用同样的方法和比你弱小的国家去打交道。对于孩子和父母的关系，银律就是：你希望你的父母如何对待你，你就要用这样的方式去对待你的孩子。

银律还可以是：不要听从那些以咨询谋生的人给你的建议，除非他们会被自己提供的错误建议受到惩罚。这实际上引出出了一个值得重视的问题：代理人问题。 所谓的代理人问题就是交易过程中因为利益不一致而导致的不对称性，例如买房或者买车这种一次性交易，对于供应商来说你不太可能是他的回头客，你和他之间的合作并是长期的，而是一锤子买卖，那么他就有可能会对你隐藏一些关键信息。

现实中有大量的职业的工作内容是以给别人建议，告诉别人什么有利，甚至有些建议是不请自来的，甚至会告诉你一切都对你有利，但事情最后往往都变成像你兜售一个产品，这个东西可以是某个消费品，可以是某个投资组合，可以是某个课程，但最后的结果往往是在达成交易后，一方享受确定性的结果而另一方承担不确定的结果，这是一种不对称性。

在当代生活里，的确充斥着不少代理人的问题，对于某一些专业化的领域，我们不得不去面对专业的人士，而在这些领域里面，有时候会充斥不少不对称的风险。以医生为例，在现代医疗体制中，医生很可能根据自己的考核指标而不是病人的实际情况来决定给病人用什么治疗方案，医生很可能会为了避免对其职业生涯造成影响的官司，而采用本来不该采用的治疗方法。就美国的医疗体系来说，这其实是一个很严重的问题，许多的过度治疗就是因为医生为了避免潜在的诉讼风险，进行了更多的医疗干预。

所以塔勒布的意思是什么呢，他认为你不应该对所有人都一视同仁，你应该对和你利益攸关的人和交易更加在乎；如果一个人对他的言论并不负责，那你就不应该把他的言论太当回事；如果一个人所在的领域缺乏风险的筛选机制，那你应该对他关于这个领域的意见要小心；如果你获得的知识没有经过足够的检验，那应该对这些知识抱有怀疑的态度，反过来如果你从实践甚至失败中获得的知识，是你珍贵的财富。
